# Command

```shell
# help
$ npx hardhat help
# compile
$ npx hardhat compile
# typechain
$ npx hardhat typechain
# deploy
$ npx hardhat run scripts/deploy.ts --network goerli
# upload code to etherscan
$ npx hardhat verify --network goerli CONTRACT_ADDRESS
# scripts
$ npx hardhat run scripts/xxx.ts
```
